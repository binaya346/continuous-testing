# Continuous Testing Course Materials

This directory contains comprehensive materials for learning and teaching **Continuous Testing** in a DevOps environment.

## 📁 Directory Structure

### 📝 [notes/](notes/)
Comprehensive educational notes on each topic:
- **01-Overview**: What is Continuous Testing (CT)?
- **02-STLC**: Software Testing Life Cycle phases.
- **03-Types of Testing**: Functional vs. Non-functional, The Pyramid.
- **04-TDD**: Test-Driven Development (Red-Green-Refactor).
- **05-Selenium**: Web automation basics.
- **06-SonarQube Setup**: Installation and administration.
- **07-SonarQube & Jenkins**: Continuous analysis integration.
- **08-SonarLint**: Real-time IDE feedback.
- **09-GH Actions Scan**: Security scanning and CodeQL.
- **10-Presentation**: Summarized class slides.

### 🛠️ [hands-on/](hands-on/)
Practical labs and scripts:
1. **[01-selenium-java-demo/](hands-on/01-selenium-java-demo/)**: A Java Maven project with a headless Selenium test.
2. **[02-jenkins-sonarqube-pipeline/](hands-on/02-jenkins-sonarqube-pipeline/)**: `Jenkinsfile` and `sonar-project.properties` for CI integration.
3. **[03-github-actions-workflow/](hands-on/03-github-actions-workflow/)**: Example `.yml` workflow for automated builds and security scans.

### 📄 [cheatsheets/](cheatsheets/)
- **[00-Testing-Cheatsheet.md](cheatsheets/00-Testing-Cheatsheet.md)**: Quick reference for Selenium commands, testing terms, and SonarQube metrics.

## 🚀 Getting Started
1. Start with the **Notes** to understand the theory.
2. Refer to the **Cheatsheet** while writing scripts.
3. Use the **Hands-on** projects to practice installation and pipeline integration.
4. Review the **Presentation** for a quick summary of the entire course.
